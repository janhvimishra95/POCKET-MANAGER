# PocketManager AI

An AI-powered Personal Finance Management System designed to help users track expenses, manage budgets, monitor savings goals, and gain intelligent financial insights.

## Features

* Expense Tracking
* Income Management
* Budget Planning
* Savings Goal Monitoring
* Financial Health Score
* Monthly Reports
* Cash Flow Forecasting
* AI-Powered Financial Assistant
* Smart Spending Analysis
* Interactive Dashboard

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API
* Chart.js
## How to Run

1. Download the project.
2. Open the folder in VS Code.
3. Install Live Server extension.
4. Right click index.html.
5. Select "Open with Live Server".
